Engineering Rhythm: A Neural Network Simulation for Muscle Coordination

Description

How does the central nervous system generate the rhythmic patterns required for locomotion? To explore this, I built a computer model of a physiological neural network known as a Unit Pattern Generator (uPG). Displayed alongside the original theoretical diagram, my dynamic animation visualizes the simulated membrane potentials of the network over time, showing how this configuration of inhibitory and excitatory connections successfully produces alternating motor output for an agonist/antagonist muscle pair, such as the quadriceps and hamstring.

Project Features

  • Computational Neuroscience: Translated a theoretical biological topology into a functional mathematical model using Python and forward Euler integration.
  • Emergent Rhythms: Demonstrated how specific inhibitory and excitatory pathways naturally produces sustained, alternating neural oscillations.
  • Visualizing Voltage: Created a visual frame-generator that translates raw membrane potential data into an intuitive color scale, capturing individual action potentials (flashes) and sustained phase shifts.
