Embedded Firmware for ECG / Temperature Medical Device

  • Engineered a thread-safe embedded system using Zephyr RTOS and a State Machine Framework (SMF) for concurrent data acquisition, processing, and BLE communication tasks.
  • Implemented real-time digital signal processing algorithms in C to filter noise and extract heart-rate from raw ECG waveforms.
  • Developed a custom BLE GATT service alongside standard Battery and Heart Rate profiles to stream sensor data and system diagnostics to a smartphone.
  • Designed a robust fault-tolerance system that detects peripheral failures (I2C/PWM/ADC), transitions the device to a safe state, and broadcasts error codes for remote debugging.
  • Optimized power consumption and memory-efficiency for battery-operated usage.
