Brain-Computer Interface (BCI) Computer Cursor Decoder

  • Developed a neural decoding algorithm in Python using a Population Vector Algorithm to translate raw motor cortex spiking activity into continuous 2D cursor movements.
  • Calculated time-binned firing rates and applied linear regression to establish the baseline, dynamic range, and preferred direction of individual neurons.
  • Simulated real-time trajectory reconstruction by taking the vector average of the neural population's normalized firing rates to iteratively update the cursor's frame-by-frame position.
  • Clinical Impact: This decoding framework serves as a foundational model for neuroprosthetics. By recording directly from the motor cortex, these algorithms can restore independence for paralyzed patients, allowing them to intuitively control external devices or computer cursors in real-time by simply imagining movement.
